# Contributing to this project

By submitting a pull request to this project, you agree to release your contribution under this project's license terms.

## How to Contribute

Before contributing, please review the [Code of Conduct](https://github.com/apple/mapkit-loader?tab=coc-ov-file).

This project follows the typical contribution process for a GitHub project, including issues and pull requests.
You can raise an issue at the repository's [issues page](https://github.com/apple/mapkit-loader/issues).
